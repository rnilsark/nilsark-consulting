# Triage context — example

Copy this to `$DOPPELGANGER_HOME/agents/triage/context.md` for private knowledge that helps the
gate decide. Injected opt-in; no file → skipped. The call sign and capability list live in
`settings.json` (structured); use this file for nuance.

## Tuning the gate

- Who is in the thread and how they usually phrase requests.
- Phrasings that look like a request but are NOT (rhetorical questions, plans already settled).
- Err toward silence on implicit messages — only escalate when the calendar/booking intent is clear.
