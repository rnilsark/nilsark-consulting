# CLAUDE.md

Project instructions for Claude Code in this repository.

## Git workflow

**Trunk-based development is allowed here.** Committing and pushing directly to `main` is fine — do
**not** create a feature branch or open a PR unless explicitly asked. (This overrides the default
"branch first on the default branch" behavior.)

Before pushing, run the relevant checks. For `doppelganger/`: `npm run typecheck` and `npm test`.
